## 自用的各类配置文件

大部分为自定义的 docker-compose.yml 配置文件，也有一些其他的（例如自己的 Clash 配置）
 
敏感信息已隐去，感兴趣可以自行取用 qwq

### Docker Compose

所有的 docker-compose.yml 都默认在当前工作目录搭建环境（例如数据卷挂载），使用时需要注意，或是自行修改相关的 volumes

部分环境变量说明：

|   环境变量    |                   说明                    |
| :-----------: | :---------------------------------------: |
| `DOCKER_USER` | 容器用户，可使用 `$(id -u):$(id -g)` 获取 |
